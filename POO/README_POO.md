# **Atividades de Programação Orientada a Objetos (POO)**

Atividades visando os principais conceitos da **Programação Orientada a Objetos**, utilizando exemplos práticos e exercícios aplicados com a Linguagem Python.

Os conteúdos apresentados nessa pasta desse repositório, envolvem uma breve introdução aos conceitos de **Estrutura de Dados** na linguagem Python, uso de tuplas, listas e dicionários.

Além de uma pasta contendo uma atividade que explorar a criação de uma **TO-DO List** que implementa as operações básicas de gerenciamento de dados conhecidas como **CRUD** (Create, Read, Update e Delete).
