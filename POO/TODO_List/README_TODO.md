# **Sistema TO-DO - Gerenciador de Tarefas**

No notebook disponibilizado nessa exata pasta, existe, em **código Python**, a simulação de um sistema **TO-DO** onde você (caso teste o código) poderá adicionar uma nova tarefa, excluir uma tarefa, mudar status dessa tarefa (para "concluída" ou "pendente), verificar sua lista de tarefas, ou simplesmente escolher sair do programa, mas sem antes receber o feedback das informações que você deixou para trás. Talvez, você pode ter deixado nenhuma tarefa restante, mas isso também será levado em consideração!
